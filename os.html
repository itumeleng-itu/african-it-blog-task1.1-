Operating Systems


Operating system: a software which performs all the basic tasks like file management, memory management, process management, handling input and output, and controlling peripheral devices such as disk drives and printers.


Directory Structure

  • Hierarchical structure starting with drives (e.g., `C:\`).
  • Key folders: `Program Files`, `Users`, `Windows`.
  • File paths: Backslashes (`\`), (e.g., `C:\Users\Name\Document.txt`).

Naming Conventions

  • Not case sensitive
  • 266 characters

File Systems

  • NTFS (default), ReFS
  • FAT32 for USB drives, SD cards

Executable File Extensions

  • `.exe` (applications)
  • `.bat` (batch files)
  • `.ps1` (PowerShell scripts)

Script Execution

  • Batch files (`.bat`): Double-click or run in Command Prompt (`cmd.exe`).
  • PowerShell (`.ps1`): Run in PowerShell (may require `Set-ExecutionPolicy`)

Directory Structure

  • Unix-like hierarchy with root (`/`).
  • Key folders: `/home` (user files), `/bin` (executables), `/etc` (configs)
  • File paths: Forward slashes (`/`), e.g., `/home/name/Document.txt`.

Naming Conventions

  • case sensitive
  • 255 characters

File Systems

  • ext4 (default), XFS, Btrfs
  • FAT32 for external media

Executable File Extensions

  • `.bin`
  • `.run`
  • `.sh`

Script Execution

  • Bash scripts (`.sh`): Execute in Terminal (`./script.sh` after `chmod +x`).
  • Shebang line: Directs OS to the interpreter (e.g., `#!/bin/bash`).
